# Dashboard

The **Dashboard** is the frontend landing page. It shows your wallet connection and network at a glance, alongside network-wide totals.

- **Connection Details** — connected wallet, network, and current block height.
- **Network metrics** — Ecosystems, Schemas, Total Locked Trust Deposit, Issued Credentials, and Verified Credentials.
- **Featured Services** — quick entry points to build your own ecosystem or discover existing ones.
