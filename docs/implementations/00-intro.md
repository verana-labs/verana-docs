# Implementations

Real, working integrations built on the Verana Trust Network. Each one takes an existing identity or credential stack and adds a verifiable trust and governance layer on top, so participants can check not just that a credential is authentic, but that the issuer is accredited and the verifier accountable.

- **[MOSIP](./mosip)** — verifiable trust for MOSIP Inji credentials. A live public pilot: real Inji Certify, Inji Verify, and Inji Web components with a Verana trust layer, running on testnet.
- **[Unfold](./unfold)** — coming soon.
